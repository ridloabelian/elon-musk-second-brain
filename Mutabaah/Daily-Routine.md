# Mutabaah Yaumiyah & Time Blocking

Keseimbangan antara ikhtiar maksimal (100 jam/minggu) dan tawakkal. Waktu sholat adalah fixed block yang gak bisa diganggu gugat.

## Daily Schedule (Time Blocking)
- **04:00 - 05:00**: Wudhu + Tahajjud + Witir (Mustajab)
- **05:00 - 06:30**: Dzikir Pagi + Subuh Berjamaah + Quran 1-2 Halaman
- **06:30 - 07:30**: Plan 3 MIT (Most Important Tasks) + Sarapan
- **08:00 - 12:00**: **Deep Work Block 1** (No Distractions)
- **12:00 - 13:00**: Dzuhur + Makan Siang + Istirahat
- **13:00 - 17:00**: **Deep Work Block 2** (Execution)
- **17:00 - 18:00**: Ashar + Review Progress + Plan Besok
- **18:00 - 19:00**: Light Work / Learning / Networking
- **19:00 - 20:00**: Maghrib + Dzikir Petang
- **20:00 - 21:00**: Isya + Quran
- **21:00 - 23:00**: Light work / Family Time
- **23:00**: Sleep (Target 5 jam malam, 1.5 jam qailulah)

## Daily Checklist
- [ ] Sholat 5 waktu on-time berjamaah di masjid
- [ ] Tahajjud (min. 2 rakaat)
- [ ] Dzikir Pagi & Petang
- [ ] Istighfar 100x
- [ ] Baca Quran (min. 1 halaman)
- [ ] Sedekah harian
- [ ] 3 MIT selesai
- [ ] Evaluasi niat bisnis (100% Halal)
